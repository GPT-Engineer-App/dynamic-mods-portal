# dynamic-mods-portal

Revamping SquawkSquad's website with a more dynamic approach, focusing on their core services of developing Minecraft mods and unique applications. This new version will emphasize a striking purple and black color scheme to align with their branding. The layout will be designed to showcase their portfolio of mods, detail their custom software solutions, and facilitate easy communication with clients. The goal is to create a user-friendly, visually appealing website that captures the essence of SquawkSquad's innovative work in software development.


This will run a dev server with auto reloading and an instant preview.
